# Loop Supervisor Roadmap

This directory captures the planned evolution of `kanban-warden` from a conservative Kanban supervisor into a traceable, verifiable, and eventually self-improving Loop Supervisor.

Read in order:

1. [v0.2 Loop Trace and Verification](v0.2-loop-trace-verification.md)
2. [v0.3 Improvement Proposals and Config Changes](v0.3-improvement-proposals.md)
3. [v0.4 Controlled Self-Improvement](v0.4-self-improvement.md)

The roadmap intentionally separates near-term observability from later self-improvement. v0.4 depends on evidence and audit records produced by v0.2 and v0.3.
